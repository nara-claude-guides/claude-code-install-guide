## 📋 시작하기 전에

- ✅ 마음의 준비: **"실수해도 괜찮아요"** 🌱

> 💡 **소요 시간**: 약 20~30분  
> 💡 **권장 모드**: Plan 모드 (사고 과정 관찰용)

## 🤔 왜 Hello World부터?

프로그래밍 세계에서 **"Hello, World!"** 출력은 50년 넘는 전통이에요. 이유가 있어요:

| 이유 | 설명 |
|------|------|
| ✅ **환경 검증** | 도구가 제대로 설치됐는지 확인 |
| ✅ **첫 성공 경험** | 작은 성취가 다음 도전의 연료 |
| ✅ **단순함** | 변수가 적어서 문제가 생겨도 원인 찾기 쉬움 |
| ✅ **공통 언어** | 전 세계 개발자가 같은 출발선에서 시작 |

> 🌱 **Fail Forward 관점**: Hello World는 **"실패해도 안전한 첫 시도"** 예요. 여기서 막혀도 괜찮아요. 막히는 지점이 곧 배우는 지점이니까요.

## 🎬 시나리오 1: 첫 대화 (말 걸어보기)

### Step 1. 작업 폴더 만들고 들어가기

```bash
mkdir hello-claude
cd hello-claude
```

### Step 2. Claude Code 실행

```bash
claude
```

화면에 입력창이 뜨면 성공! 🎉

```
🤖 Claude Sonnet · /Users/nara/hello-claude · main
─────────────────────────────────────────────────
> _
```

### Step 3. 첫 인사 건네기

입력창에 이렇게 써보세요:

```
안녕! 너랑 첫 대화야. 자기소개 한번 해줘.
```

### 🎁 Claude의 응답 (예시)

```
안녕하세요! 저는 Claude예요. Anthropic이 만든 AI 코딩 어시스턴트입니다.

이 폴더(/Users/nara/hello-claude)에서 작업할 준비가 되어 있어요.
파일을 읽고, 만들고, 수정하고, 명령어도 실행할 수 있어요.

뭘 만들어볼까요?
```

### 👀 여기서 주목할 것

- 🤖 Claude가 **현재 폴더 경로를 알고 있어요** → 이게 "컨텍스트"
- 💬 평범한 한국어로 자연스럽게 대화 가능
- 📋 도구를 사용하지 않은 일반 대화는 `⏺` 표시 없이 답변만

> 💡 **놀라운 점**: 영어 명령어를 외울 필요 없어요. **"~해줘"** 만으로도 충분해요!

## 🎬 시나리오 2: 첫 파일 만들기

이제 진짜 Hello World를 만들어봐요!

### Step 1. 파일 만들기 요청

```
hello.py 라는 파일을 만들어줘. "Hello, World!"를 출력하는 간단한 파이썬 코드로.
```

### 🎁 Claude의 응답 (예시)

Claude가 도구를 사용하기 시작해요:

```
hello.py 파일을 만들어드릴게요.

⏺ Write(hello.py)
  ⎿ 파일을 만들었습니다 (1줄)

다음 내용으로 hello.py를 생성했어요:

print("Hello, World!")

실행하려면 터미널에서 python hello.py 를 입력하시면 됩니다.
```

### ⚠️ Default 모드라면 권한 다이얼로그가 뜰 거예요

```
┌─────────────────────────────────────────┐
│  Claude wants to:                       │
│      Write(hello.py)                    │
│                                         │
│  [ Y ] Yes, run once                    │
│  [ A ] Yes, and don't ask again         │
│  [ N ] No                               │
└─────────────────────────────────────────┘
```

→ **Y**를 눌러 허용해보세요!

### Step 2. 파일이 진짜 생겼는지 확인

```
!ls -la
```

> 💡 입력창에 `!`로 시작하면 **셸 명령어를 직접 실행**해요!

```
total 8
drwxr-xr-x  3 nara  staff   96  4 27 14:30 .
drwxr-xr-x  5 nara  staff  160  4 27 14:30 ..
-rw-r--r--  1 nara  staff   23  4 27 14:30 hello.py
```

🎉 **`hello.py`가 보이네요!**

### 👀 여기서 주목할 것

- 📂 Claude는 진짜로 **파일 시스템을 만질 수 있어요**
- 🔐 Default 모드에서는 **매 작업마다 허락**을 구해요 (안전장치)
- 📝 코드도 같이 보여줘서 무엇을 만들었는지 투명하게 알 수 있어요

## 🎬 시나리오 3: 코드 실행해보기

만든 코드가 진짜 동작하는지 봐볼까요?

### Step 1. 직접 실행 요청

```
hello.py 를 실행해줘.
```

### 🎁 Claude의 응답 (예시)

```
실행해보겠습니다.

⏺ Bash(python hello.py)
  ⎿ Hello, World!

성공적으로 실행되었어요! "Hello, World!"가 출력되었습니다. 🎉
```

### 🎉 축하해요! 첫 코드를 만들고 실행하기까지 성공!

### 🤔 만약 에러가 났다면?

Python이 설치 안 되어 있을 수 있어요:

```
⏺ Bash(python hello.py)
  ⎿ command not found: python
```

이럴 때 Claude에게 그냥 말해보세요:

```
python이 없다는데, python3로 다시 해볼래?
```

→ Claude가 알아서 `python3 hello.py`로 시도할 거예요!

> 🌱 **Fail Forward**: 에러가 났다고 멈추지 마세요. **에러 메시지를 그대로 Claude에게 보여주는 것**이 가장 빠른 해결책이에요.

## 🎬 시나리오 4: 다른 언어로 만들어보기

같은 Hello World를 여러 언어로 만들어보면 패턴이 보여요.

### Step 1. JavaScript로

```
같은 Hello World를 JavaScript로 hello.js 파일로 만들어줘.
```

**Claude의 결과:**

```javascript
console.log("Hello, World!");
```

### Step 2. HTML로

```
이번에는 hello.html 파일로 만들어줘. 브라우저로 열어서 볼 수 있게.
```

**Claude의 결과:**

```html
<!DOCTYPE html>
<html>
<head>
  <title>Hello World</title>
</head>
<body>
  <h1>Hello, World! 👋</h1>
</body>
</html>
```

### Step 2-1. 현재 디렉토리에서 웹서버 실행

```
지금 런박스 환경인데 브라우저로위의 파일을 열어 보는 방법 없어?    
```

**Claude의 결과:**

```● 네! 간단한 HTTP 서버를 띄워서 접근할 수 있습니다. 현재 디렉토리에서 웹   
  서버를 시작해보겠습니다:                                                 
                                                            
● Bash(python3 -m http.server 8000)             
  ⎿  Running in the background (↓ to manage)                               
                                                            
● 완료했습니다! HTTP 서버가 백그라운드에서 실행 중입니다.                  
                                                                           
  이제 다음 URL로 접근할 수 있습니다:                       
  - http://localhost:8000/hello.html                                       
                                                                           
  runbox 환경에서:
  1. Claude Code의 Preview 기능이 있으면 위 URL을 입력하여 미리보기 가능   
  2. 또는 터미널에서 curl 또는 wget으로 확인:                              
  curl http://localhost:8000/hello.html                                    
                                                                           
  파일을 변경하면 자동으로 반영되니까 편하게 사용할 수 있어요! 
```

### Step 3. 한 번에 비교 부탁하기

```
지금까지 만든 hello.py, hello.js, hello.html 의 공통점과 차이점을 표로 정리해줘.
```

### 🎁 Claude의 응답 (예시)

| 항목 | hello.py | hello.js | hello.html |
|------|----------|----------|------------|
| 언어 | Python | JavaScript | HTML |
| 출력 명령 | `print()` | `console.log()` | `<h1>` 태그 |
| 실행 방법 | `python hello.py` | `node hello.js` | 브라우저로 열기 |
| 용도 | 백엔드, 자동화 | 웹·앱 개발 | 웹 화면 |

### 👀 여기서 주목할 것

- 🧠 Claude가 **여러 파일을 한꺼번에 기억**하고 있어요 (이게 컨텍스트)
- 📊 표 형태도 자연스럽게 만들어줘요
- 🌐 같은 결과를 **여러 가지 방법**으로 만들 수 있다는 걸 알게 돼요

## 🎬 시나리오 5: 살짝 응용해보기

이제 단순한 Hello World를 넘어볼까요?

### Step 1. 이름 받기로 변경

```
hello.py를 수정해줘. 사용자 이름을 입력받고, "Hello, [이름]!" 으로 출력하게.
```

**Claude의 결과:**

```python
name = input("이름을 입력하세요: ")
print(f"Hello, {name}!")
```


### Step 2. 직접 실행해서 이름 넣어보기

```
!python hello.py
```

```
이름을 입력하세요: 신나라
Hello, 신나라!
```

### Step 3. 더 재밌게 만들기 부탁

```
시간대에 따라 인사말이 바뀌게 해줘. 아침엔 "좋은 아침", 오후엔 "안녕하세요", 밤엔 "좋은 밤" 이런 식으로.
```

### 🎁 Claude의 응답 (예시)

```python
from datetime import datetime

name = input("이름을 입력하세요: ")
hour = datetime.now().hour

if 5 <= hour < 12:
    greeting = "좋은 아침"
elif 12 <= hour < 18:
    greeting = "안녕하세요"
elif 18 <= hour < 22:
    greeting = "좋은 저녁"
else:
    greeting = "좋은 밤"

print(f"{greeting}, {name}님!")
```

### 👀 여기서 주목할 것

- 🔄 코드를 **점진적으로 발전**시킬 수 있어요
- 💡 **"이렇게 해줘"** 만 말해도 알아서 코드 구조를 잡아줘요
- 🧪 학습자가 직접 만들기 어려운 기능도 **요청만으로 시연**할 수 있어요

## 🆘 자주 막히는 지점

### Q1. 입력창이 안 뜨고 멈춰요!

```bash
# 현재 어떤 상태인지 확인
/status

# 그래도 막히면 한 번 종료 후 재시작
Ctrl+D
claude
```

### Q2. 권한을 잘못 눌러서 거부했어요

괜찮아요! 그냥 다시 요청하면 돼요:

```
방금 거부한 작업 다시 해줘
```

### Q3. Claude가 다른 파일을 건드릴까봐 무서워요

**Plan 모드**로 시작하세요! 모든 작업을 사전에 보여주고 승인받아요:

```bash
claude --permission-mode plan
```

또는 세션 내에서 `Shift+Tab`을 눌러 모드 전환.

### Q4. "command not found: python" 에러

```
python3 으로 다시 해줘
```

또는 Claude에게:

```
python이 설치되어 있는지 확인하고, 없으면 어떻게 설치해야 하는지 알려줘
```

### Q5. 한국어가 깨져 보여요

터미널 인코딩 문제일 수 있어요:

```bash
# zsh/bash 설정에 추가
export LANG=ko_KR.UTF-8
export LC_ALL=ko_KR.UTF-8
```

### Q6. Claude가 너무 길게 답해요

```
앞으로는 짧고 핵심만 말해줘
```

또는 시스템 프롬프트로 시작:

```bash
claude --append-system-prompt "답변은 핵심만 간결하게 한국어로"
```
